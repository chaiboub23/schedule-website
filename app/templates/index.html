How this site works

This site uses a Webcal link from your RNS Portal and stores this link into a cookie. This cookie is then fetched by the Flask server and Python is used get the iCalendar data from the Webcal url. This is then parsed and converted into a Python list containing your schedule for the day and a Python dictionary containing your assignments.

Help

To get the url, login to the RNS Portal

Then go to the Calendar section

Click on the symbol that looks like the WiFi symbol

Right click on 'My Calendars' and copy the link

Paste this link into the text box above
